# Decisions

This folder contains long-lived architectural and design decisions for the project.

## Format

Decisions are stored as numbered Markdown files (e.g. `0001-*.md`).

Each decision should include:
- Context / problem
- Decision
- Consequences
- Date
- Status (proposed / accepted / superseded)

## Examples

- `0001-episode-pages.md` — Design decisions made while building the Episode Pages feature.

## Relationship to AI tooling

Temporary AI planning artifacts (PRDs, session notes, agent configurations) live in `.agents/` and are **not** committed to the repository.

This folder is reserved for decisions that have lasting value to human contributors.
